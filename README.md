# Create vbapp

`` create-vbapp `` is a command line tool to create projects with vbapp.

## Usage

```bash
# Interactive mode
pnpm create-vbapp

or 

npx create-vbapp
```

## Installation

```bash
pnpm install -g create-vbapp
```
